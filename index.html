Store Defaults

Smart Sales Assistant

When ON: unknown messages are answered by Mistral AI using your live product catalog. When OFF: only local price/stock/order replies are used.

Used only if Mistral is disabled or unavailable.

WhatsApp Group Auto-Reply

When ON: the AI chatbot will also reply to messages in WhatsApp groups.
When OFF (default): chatbot only replies in direct/personal chats.

Enter group names (partial match) or full group JIDs, one per line. Leave blank to reply in all groups.

End-of-Day WhatsApp Report

Automatically sends a daily sales summary to a WhatsApp number every evening — revenue, top products, expenses and low-stock alerts.

Leave phone blank to use the Business Phone from the Business tab. Uses your timezone setting.

WhatsApp Anti-Spam Guard

Rate-limits, duplicate detection, keyword blocking and auto-ban protect the bot from abuse.

Keywords are case-insensitive. Messages containing any keyword are flagged as spam.

Daily GitHub Backup

Automatically commits a JSON snapshot of all your data (products, customers, invoices, etc.) to a private GitHub repo every day. Restore anything in 30 seconds — even if your hosting wipes the disk.

Create at github.com/settings/tokens → Fine-grained token → only the backup repo → permissions: Contents: Read & Write.

Format: owner/repo-name. Make it private. Repo must already exist (any default branch is fine).

Default 0 3 * * * = every day at 3:00 AM. Use crontab.guru to customise.
